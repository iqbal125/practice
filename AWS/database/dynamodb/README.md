# DynamoDB

Fully managed NoSQL database service providing fast and predictable performance with seamless scalability. Supports both key-value and document data models with built-in security, backup, and in-memory caching.





Time To Live (TTL) for DynamoDB allows you to define when items in a table expire so that they can be automatically deleted from the database.
